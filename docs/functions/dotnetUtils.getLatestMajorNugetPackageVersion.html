Function getLatestMajorNugetPackageVersion

  • getLatestMajorNugetPackageVersion(packageName, targetFrameworkMoniker): Promise<null | number>

  • Same as getLatestNugetPackageVersion, except it first pulls from a hard-coded list, and only if it isn't found will it reach out to nuget.org to screen scrape and use their best guess. Only returns the major version - use it to import a package with wildcard syntax, for example: dotnet add package SomePackage -v 7.*.

    Useful for packages like the EntityFramework that is technically compatible with versions of .net that they actually don't work with because of runtime dependencies of the dotnet-ef tool.
